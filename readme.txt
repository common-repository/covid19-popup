== Description ==
This is a very simple plugin with the sole purpose of allowing website owners a quick way to add a COVID-19 notice to their website. This free plugin is intended to provide a stop-gap solution and does not attempt to compete with more full-featured "banner" plugins. 

This plugin now offers additional data display features by using the shortcode <code>[covid19_popup]</code>. See Screenshot to see how it works.

In v2 you don't need to use shortocde for display the data. Actually we have big changed with this plugin, there are too many options to display data and also you can change the appreance.

we are using <a href="https://covid19api.com/">COVID19API</a> apis in our plugin. 
So plugin functionality and data totally depends on third party APIs. (We can’t provide any support if there is any downtime in third party APIs).
For more information and Please check <a href="https://covid19api.com/">COVID19API</a>
Please check  PRIVACY POLICY.
https://covid19api.com/privacy-policy.html


== Installation ==
FROM YOUR WORDPRESS DASHBOARD
1. Visit ‘Plugins > Add New’
2. Search for ‘covid19-popup’
3. Activate covid19-popup from your Plugins page.

OR BY USING THE FOLLOWING STEPS
1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= Who would use this plugin? =

WordPress websites that need to display notices regarding COVID-19, or those who want to display the latest COVID-19 data (confirmed, deaths).

= How to display on the website? =
 
In v2 you don't need to use shortocde for display the data. Actually we have big changed with this plugin, there are too many options to display data and also you can change the appreance. Use This shortcode <code>[covid19_popup]</code> to display global data and <code>[covid19_popup country="india"]</code> to display specific country data.
